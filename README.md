# Start back end Spring Boot application:
On main directory of the project
  #First init:
  ```mvn package ```
  To build maven packages
  
  #Run spring boot application:
  ``` mvn spring-boot:run ```
  
# Start front-end React application:
  Go to directory (from main project's directory):
  ``` cd src/front-end/my-app ```
  #First init:
  ``` npm install ``` 
  To install npm packages
  
  #Run React container:
  ``` npm start ```
